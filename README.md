## 22년도 1학기 수강가이드
- **방향:** 위 교과목내용 변경계획으로 22년도 1학기 **`E데이터베이스`** 와 **`빅데이터`** 수업은     
데이터비즈니스에 늦게 입문 또는 첫 입문한 수강생을 대상으로 하기에 동일내용이며,     
부득이하게 기 입문한 21년도 2학기 1학년 대상 **`E정보시스템`** 강의 내용과 60% 정도 겹칩니다.
